# **App Name**: InsightForge

## Core Features:

- Video Insights Display: Display video insights metrics such as views, likes, comments, shares, reach, impressions, and average watch time.
- Date Range Selection: Allow users to select a date range (Last 7 days, 30 days, Custom) to view insights for a specific period.
- Inline Metric Editing: Enable users to manually override metric values (views, date) with inline editing.
- Data Visualization: Visualize video insights using line charts for views over time and bar charts for top countries.
- Insight Manipulation: Duplicate an existing insight, or apply mock demo data using a tool to create test cases or presentations.
- CSV Export: Export insight data, metrics and history as a CSV file for further analysis and reporting.
- Data Storage: The video data will be stored in the database for persistent storage, analysis, and recall. Use Firebase for quick MVP.

## Style Guidelines:

- Primary color: Saturated blue (#3B82F6) to inspire confidence and signal a data-driven application.
- Background color: Light blue (#E0F7FA), providing a clean and professional look with minimal distraction.
- Accent color: A vibrant violet (#8B5CF6) to highlight interactive elements, providing a contrast to the primary and background colors and a feeling of modern insight.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined and neutral aesthetic.
- Use minimalist icons to represent metrics, keeping the UI clean and intuitive.
- Employ a clean, card-based layout with rounded corners and subtle shadows to create depth.
- Incorporate subtle animations for loading states and transitions to enhance the user experience.
